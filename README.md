Сборка Android-приложения
Откройте проект PhoneBatterySync в Android Studio.

Убедитесь, что в build.gradle подключен Wear Engine SDK (com.huawei.hms:wearengine:6.1.0.301).

Соберите APK: Build -> Build Bundle(s) / APK(s) -> Build APK(s).

Установите на телефон (поддерживающий сервисы HMS) и запустите. Нажмите кнопку «Запустить сервис» для активации foreground-сервиса.

Сборка и подпись приложения для часов (HarmonyOS)
Установите DevEco Studio для разработки под HarmonyOS (версия 3.0+).

Импортируйте проект BatteryWatchApp.

В файле build.gradle укажите зависимости Wear Engine для HarmonyOS:

text
dependencies {
    implementation 'com.huawei.hms:wearengine:6.1.0.301'
}
Настройте подпись приложения:

Создайте сертификат и профиль подписи через Huawei AppGallery Connect.

В DevEco Studio выберите Build -> Generate Key and CSR, затем загрузите в AppGallery Connect для получения сертификата выпуска.

Укажите файлы .p12 и .cer в File -> Project Structure -> Signing Configs.

Соберите HAP: Build -> Build HAP(s).

Установите на часы через adb или путём копирования в память часов и запуска установщика.

Использование
После установки на телефон и часы откройте приложение «Заряд телефона» на часах.

При запуске автоматически отправляется запрос. Если телефон в зоне действия и foreground-сервис активен, на часах появится процент заряда и статус.

Нажмите «Обновить» для повторного запроса.

Если телефон не отвечает в течение 3 секунд, выводится сообщение «Телефон не отвечает». Повторите позже.

Примечание
Телефонное приложение работает только в паре с часами, поддерживающими Wear Engine (Huawei Watch GT 4).

Убедитесь, что на телефоне не отключены фоновые процессы для этого приложения (отключена оптимизация батареи по запросу).
